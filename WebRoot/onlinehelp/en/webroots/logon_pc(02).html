1. (Optional) Install the cloud client at the first login. Installation method: Open the browser of a PC, enter the cloud server address in the address box and press EnterÂ to go to the login page. On the login page, enter the user domain username and password, and then click click here to install the cloud client.

2.  Choose Start to open the cloud client (or double-click the shortcut of the cloud client on the desktop to start it), and configure cloud server address.

3.  On the login page, enter the domain username and password.

4.  On the displayed VM list page, click VM to enter the VM desktop.
