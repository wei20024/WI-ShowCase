Personalized VM desktop settings

1.   Set the VM resolution.

    The VM's resolution will accordding to the TC/PC. You need to change it on the TC/PC.

2.  Set the VM mouse or keyboard.

     When you set the mouse or keyboard property on the TC/PC, the property on the VM will be affected.
     When you set the mouse or keyboard property on the VM, the property only on the VM will be affected.

3.  Set the VM volume.  Any of the following items can affect the VM volume:

     TC/PC volume            VM volume            Third-party application volume

      The noisy environment will affect the voice call. You can reduce the microphone volume of the TC.

      Ensure that only one client is enabled on a TC. When multiple clients are enabled at the same time, the         voice call may be affected. In this way, you are advised to restart the TC and then enable one client.
